# Tiny PDF Editor v1.0.1

Windows용 **포터블 PDF 편집기**입니다. Python 설치 없이 실행할 수 있는 배포판을 제공하며, 페이지 병합·편집·검색·용량 줄이기 등 일상적인 PDF 작업을 한 프로그램에서 처리할 수 있습니다.

## 다운로드 및 실행 (배포판)

1. 블로그 또는 배포 페이지에서 **최신 빌드 폴더 전체**를 받습니다.
2. 압축을 푼 뒤, 폴더 안의 `Tiny PDF Editor v1.0.1_YYMMDD_HHMMSS.exe`를 실행합니다.
3. `_internal` 폴더와 exe는 **같은 위치**에 있어야 합니다. exe만 따로 복사하면 실행되지 않습니다.

USB에 폴더 전체를 복사해 다른 PC에서도 사용할 수 있습니다. 자세한 내용은 `DISTRIBUTE.md`를 참고하세요.

### 시스템 요구 사항

- Windows 10 이상 (64비트)
- 인터넷 연결 불필요 (오프라인 사용 가능)

> Windows Defender 등에서 처음 실행 시 경고가 나올 수 있습니다. 직접 빌드하거나 신뢰하는 출처의 배포본이라면 「추가 정보」→「실행」을 선택하세요.

## 주요 기능

### 파일 · 탭

- **열기 / 저장 / 다른 이름으로 저장**
- **인쇄** (Windows 기본 PDF 인쇄)
- **탭**: 여러 PDF 문서를 동시에 열기

### 썸네일 (왼쪽)

- **드래그 앤 드롭**: PDF 또는 이미지를 끌어다 놓아 해당 위치에 페이지 삽입
- **다중 선택**: `Ctrl` + 클릭, 또는 빈 공간 드래그
- **삭제**: `Delete` 키, 휴지통 버튼, 우클릭 메뉴
- **회전**: 시계/반시계 방향
- **페이지 보내기**: 선택 페이지만 새 PDF로 저장
- **이미지로 보내기**: 선택 페이지를 PNG/JPEG로 저장
- **썸네일 크기**: `+` / `-` 버튼으로 조절

### 본문 보기 (오른쪽)

- 선택 페이지 미리보기
- **너비/높이 맞추기**, **확대/축소** (슬라이더, `Ctrl` + 마우스 휠)
- **페이지 이동**: 맨 앞 / 이전 / 다음 / 마지막
- 페이지 크기(cm) 표시

### 편집

- **텍스트 검색** (`Ctrl+F`, `F3` / `Shift+F3`): 본문 텍스트 검색 및 하이라이트
- **용량 줄이기** (`편집` 메뉴): PDF 파일 크기를 줄이는 도구 (아래 참고)

### 용량 줄이기

`편집` → **용량 줄이기...** 메뉴에서 사용합니다. 텍스트는 유지하고 이미지 위주로 압축합니다.

**기본 설정**

| 항목 | 기본값 |
|------|--------|
| 이미지 사이즈 | 50% |
| 이미지 품질 | 50% |
| 최대 DPI 제한 | 150 DPI |
| 희망 최종 용량 (2단계) | 9.8 MB |

**사용 순서**

1. **[1단계] 예상 최종 크기 산정**  
   현재 설정으로 압축했을 때의 예상 파일 크기를 계산합니다.  
   산정이 끝나면 **[최종 적용 하기]**로 바로 문서에 반영할 수 있습니다.

2. **[2단계] (선택) 희망 최종 용량 설정 및 용량 줄이기 자동 반복**  
   원하는 목표 용량(MB)을 정한 뒤 **[용량 줄이기 반복 실행]**을 누르면, 목표 용량 미만이 될 때까지 설정을 자동 조정·재산정합니다.  
   2단계 없이 1단계 결과만 적용해도 됩니다.

3. **[최종 적용 하기]**  
   산정된 결과를 현재 문서에 적용합니다. 적용 후 **저장** 또는 **다른 이름으로 저장**으로 파일을 저장하세요.

> 스캔 PDF(JBIG2 등)처럼 일부 이미지는 처리에서 건너뛸 수 있습니다. 문제가 있는 이미지는 원본을 유지하고 나머지를 계속 처리합니다.

### 지원 파일 형식 (삽입)

- PDF (`.pdf`)
- 이미지: PNG, JPEG, BMP, GIF, TIFF, WebP

## 개발자용: 소스에서 실행

```bash
pip install -r requirements.txt
python main.py
```

### Windows 배포판 빌드

Node.js가 설치되어 있어야 합니다.

```bash
npm install
npm run build:dist:exe
```

증분 업데이트(최신 `dist` 폴더에 변경 파일만 반영):

```bash
npm run build:dist:exe:update
```

빌드 결과는 `dist/` 아래에 생성됩니다.

```
dist/
  Tiny PDF Editor v1.0.1_YYMMDD_HHMMSS/
    Tiny PDF Editor v1.0.1_YYMMDD_HHMMSS.exe
    _internal/          (실행에 필요한 라이브러리)
    LICENSE
    README.md
    DISTRIBUTE.md
```

빌드 폴더는 최근 **3개**만 유지됩니다.
